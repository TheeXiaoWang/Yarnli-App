## ðŸ”§ Development Workflow

### Standard Workflow
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write/update tests
5. Run `npm run ci` to verify
6. Submit a pull request

### Before Committing
```bash
npm run ci  # Runs all checks
```

This runs:
- Type checking
- Linting
- Format checking
- Tests with coverage
- Production build
